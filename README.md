## 프로젝트 개요
**본프로젝트**에서는 **Chat-GPT4**에게 파이썬에 내장된 **파이게임**을 활용한 태양계의 행성을 표현하는 프로그램을 제작 및 수정을 요청하여 코드를 받았습니다. <br>
<br><br>

## 게임설명 및 실행
### 1️⃣ 태양계
태양계의 행성의 움직임을 표현
<br><br>

##### 게임 설명
- **조작법** : 시간의 속도를 조정해 각각의 행성의 위치를 본다.
- **특징** : 처음 프로그램을 시작하면 태양계 행성들이 일렬 정렬이 된 상태로 시작한다.
- **주의사항** : 지구시점이 아닌 태양계 전체의 시점이다.
  
<br>
