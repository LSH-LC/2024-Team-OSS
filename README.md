## 프로젝트 개요
**본프로젝트**에서는 **Chat-GPT4**에게 파이썬에 내장된 **파이게임**을 활용한 태양계를 표현하는 프로그램을 제작 및 수정을 요청하여 코드를 받았습니다. <br>
<br><br>

## 게임설명 및 실행
### 1️⃣ 태양계
태양계를 시간에 따라 변화하는 모습을 표현
<br><br>

##### 게임 설명
- **조작법** : 시간을 조정해 원하는 시간대를 찾는다.
- **특징** : 특정시간의 각 행성의 위치를 찾는다.
- **주의사항** : 지구시점이 아닌 태양계 전체의 시점이다.
